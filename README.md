# DevTools Site Cloner

DevTools Site Cloner is a Python GUI tool that allows you to clone entire websites, including images, CSS, JavaScript, and fonts. It uses PyQt5 for the user interface, Playwright for rendering and interacting with pages, and aiohttp for asynchronous resource downloading.

## Features

- User-friendly GUI with PyQt5
- Selective resource downloading (Images, CSS, JS, Fonts)
- Recursive site crawling with configurable depth
- Live progress tracking and stop functionality

## Installation

```bash
git clone https://github.com/yourusername/site-cloner.git
cd site-cloner
pip install -r requirements.txt
playwright install
```

## Usage

```bash
python main.py
```
